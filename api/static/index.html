Pose Processing API

Welcome to the Pose Processing API. Use the following endpoints to interact with the API:

GET /api/session

Get the current session or create a new one

POST /api/upload

Upload a video file for processing

GET /api/files

Get files for the current session

POST /api/process

Process a video with selected models and filters

GET /api/process/available-models

Get a list of available models and filters

GET /api/jobs/{job_id}

Get the status of a specific job

GET /api/jobs/{job_id}/result

Get the results of a completed job

GET /api/info

Get API information and status

For more information, please check the documentation.
